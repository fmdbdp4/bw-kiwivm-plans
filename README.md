## 搬瓦工怎么样：线路速度实测结论

速度这个问题，要分线路说，不能一概而论。

KVM 套餐走的是普通国际线路，延迟大概在 150–200ms 区间，晚高峰会有些波动，适合预算有限或者不介意速度的场景。它已经把 DC3 CN2 机房从可切换列表中移除了，所以现在的 KVM 套餐走的就是普通线路，别指望它能有 CN2 的速度。

CN2 GIA-E 套餐完全是另一回事。根据多个第三方测评站点（VPS GO、P3TERX 等）的实测数据，DC6 CN2 GIA-E 机房的电信延迟大约在 140–160ms，联通和移动也能走优化路线，晚高峰表现明显优于普通线路。日本软银（JPOS_1）机房对联通用户格外友好，延迟有时能压到 100ms 以内。

香港 CN2 GIA 就是目前搬瓦工速度天花板。知乎上有用户测试显示，香港节点的延迟可以低至 50ms 以下。但价格也是天花板级——$89.99/月，所以真正买的人很少，更多是作为极致体验的备选。

一个实用的判断方式：如果你对速度有要求，先别买最便宜的，季付 $49.99 的 CN2 GIA-E 是大多数国内用户最均衡的选择，比年付 $49.99 的 KVM 套餐每个月多花不了多少，但线路质量差距可以很明显。
